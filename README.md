# Hello Health App


##Description
Hello Health App is a simple python program created created as a paert of Oblig 1. The purpose of the project is to practice basic python structure, git usage, and Github delivery.

## Installation Instruction


1. Clone repository:
  ```bash 
   git clone https://github.com/your-username/your-username-oblig1.git

2. Navigate into project directory:
  ```bash
   cd teldavid05-prog-Oblig1

3. create virtual enviroment 
   ```bash
   python3 -m venv .venv

4. Activate the virtual enviroment:
   ```bash
   source .venv/bin/activate

5. Run the application
   ```bash
   python hello.health.py

Requirements:

python 3.10 or higher
no external dependencies for Oblig 1
